Clasificación Binaria de Imágenes con Random Forest y KNN: Un Estudio Comparativo

IA Clasificación

Este proyecto aplica algoritmos de clasificación supervisada para analizar imágenes, primero con el dataset Flavia (hojas) y luego con MNIST (dígitos), enfocándose en identificar el número 7. Se compararon los modelos Random Forest y KNN mediante diversas métricas de evaluación. En ambos casos, Random Forest mostró un mejor desempeño, destacándose por su precisión y capacidad de generalización en distintos tipos de datos visuales.

Resultados

  • Random Forest superó a KNN en precisión, recall y AUC en ambos experimentos (Flavia y MNIST).
  • La curva ROC del modelo Random Forest en MNIST arrojó un AUC de 0.99, indicando una alta capacidad de discriminación.
  • Ambos modelos fueron evaluados con validación cruzada, garantizando resultados confiables y replicables.

Análisis Predictivo del Alquiler de Bicicletas mediante Técnicas de Aprendizaje Automático

Regresión supervisada Aprendizaje automático

Este estudio aplica modelos de aprendizaje automático supervisado para predecir la demanda de alquiler de bicicletas con base en variables ambientales y temporales. Se emplean técnicas como regresión lineal, árboles de decisión y K-Nearest Neighbors, evaluando su rendimiento para determinar el modelo más preciso.

Resultados

  • El modelo KNN con 4 vecinos fue el más preciso para predecir el total de bicicletas alquiladas, presentando menor error cuadrático medio y mayor puntuación R² frente a otros modelos probados.

Machine Learning para Predecir el Transporte de Pasajeros en el Titanic Spaceship

ML RandomForest

Este proyecto explora el uso de modelos de machine learning para predecir la probabilidad de que un pasajero de la nave espacial "Titanic Spaceship" haya sido transportado o no durante una anomalía en su viaje interestelar. Se utilizaron técnicas de análisis exploratorio, limpieza de datos y algoritmos de clasificación como Random Forest y Gradient Boosting, empleando un dataset simulado con atributos como el planeta de origen, cabina, edad, y gasto en distintas áreas de la nave. El objetivo principal fue construir un modelo predictivo eficaz y evaluar su rendimiento mediante métricas como accuracy y F1-score.

Resultados

  • El modelo de Gradient Boosting obtuvo una precisión superior al 80% en el conjunto de prueba, superando a otros algoritmos como KNN y SVC.
  • Variables más influyentes: Las características como HomePlanet, CryoSleep, VIP, y RoomService se identificaron como las más relevantes para determinar si un pasajero fue transportado.
  • La imputación de valores nulos mediante estrategias basadas en la media y el modo permitió mejorar significativamente la calidad del dataset sin introducir sesgos.

DeepPredict: Modelo de predicción de Cáncer de Mama mediante Inteligencia Artificial en Imágenes Histológicas

DL Cáncer de Mama

Este proyecto presenta un sistema de Deep Learning para la clasificación de imágenes histológicas de cáncer de mama. Utilizando la arquitectura EfficientNetB0 y técnicas de Transfer Learning, el modelo clasifica parches de imágenes como benignos o malignos. Se abordó el desbalance de clases con SMOTE y se evaluó el rendimiento con métricas clave como F1-Score y AUC. Se desarrolló una interfaz Gradio para su aplicación práctica como herramienta de apoyo diagnóstico.

Resultados

  • El modelo EfficientNetB0 optimizado con Keras Tuner fue el más efectivo, logrando un excelente recall en la detección de cáncer de mama en imágenes histológicas. Sus métricas clave demuestran su robustez y potencial como herramienta de apoyo diagnóstico.

Clustering de Recetas para Optimización de Tráfico(Tasty Bites)

ML Clustering

Este proyecto aplica técnicas de análisis de datos y clustering no supervisado (K-Means) para segmentar recetas según sus características nutricionales y categorías, con el objetivo de entender patrones de tráfico en el sitio Tasty Bytes. El análisis permite identificar grupos de recetas con potencial para atraer más usuarios, sirviendo de base para decisiones estratégicas de contenido.

Resultados

  • Se identificaron 4 segmentos de recetas bien definidos mediante K-Means.
  • Cada segmento mostró perfiles nutricionales y categóricos distintos (ej. alto en azúcar, alto en proteína, comidas rápidas, etc.).
  • Se logró una base sólida para estrategias de contenido, priorizando recetas con mayor potencial de popularidad.

Detección de Objetos Personalizada con YOLOv5

Visión por Computadora DL

Este proyecto se enfoca en entrenar un modelo de detección de objetos YOLOv5 (versión mediana) para identificar drones en imágenes. Utiliza un conjunto de datos personalizado de 526 imágenes de drones, divididas en conjuntos de entrenamiento, validación y prueba. El proceso incluye la preparación del conjunto de datos, el entrenamiento del modelo y la realización de inferencias para evaluar su rendimiento.

Resultados

  • El modelo YOLOv5 entrenado demostró la capacidad de detectar drones en imágenes, con métricas de rendimiento (mAP@0.5 y mAP@0.5:0.95) que indican una detección efectiva. Se logró un mAP@0.5 de 0.941 y un mAP@0.5:0.95 de 0.589 después de 100 épocas de entrenamiento.

Transfer Learning

Aprendizaje Automático KNN

Este proyecto explora el uso de Transfer Learning para tareas de clasificación de imágenes, utilizando modelos pre-entrenados como EfficientNetV2-B0. Se demuestra cómo cargar y adaptar estos modelos para nuevos conjuntos de datos, optimizando el proceso de entrenamiento y aprovechando el conocimiento adquirido de grandes datasets como ImageNet.

Resultados

  • El proyecto demuestra la implementación exitosa de Transfer Learning, logrando una clasificación efectiva de imágenes con modelos pre-entrenados. Aunque no se proporcionan métricas numéricas específicas en el notebook, la estructura del código y el uso de técnicas de aumento de datos sugieren un rendimiento optimizado.
